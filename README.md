Opencv3 for Android library

## Using in your Android project

Add Bintray JCenter repository:

```
allprojects {
    repositories {
        google()
        jcenter()
    }
}
```

Add dependencies (you can also add other modules that you need):

```
implementation 'com.github.xuybin:opencv3:${version}@aar'
```
